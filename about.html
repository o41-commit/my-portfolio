BIO

I’m Fatoba Victor — a frontend developer and designer who turns ideas into high-impact websites and interactive apps. With a strong focus on clean code, responsive design, and user-friendly interfaces, I blend creativity and logic to deliver digital experiences that look as good as they perform.

Frontend Developer

I specialize in building responsive, interactive user interfaces using HTML, Tailwind CSS, and JavaScript. I care about clean code, smooth UX, and modern web design. While frontend is my core, I’m also exploring backend fundamentals with Node.js to better understand how everything connects behind the scenes.

React Developer

I enjoy using React to build dynamic, interactive web apps that respond in real time. From managing state to building reusable components, I aim to write clean, maintainable code that scales.

Graphic Designer

With a strong foundation in visual design, I bring a creative edge to my development work. From branding to UI/UX mockups, I use tools like Figma and Photoshop to design layouts that are both beautiful and user-focused.
